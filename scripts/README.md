# スクリプト

## 概要

- 作業効率化のため、同一の用途のコマンドをスクリプトにまとめる

## 一覧

| ファイル名          | 概要                                | 備考 |
| ------------------- | ----------------------------------- | ---- |
| run_in_local_env.sh | ローカルで疎通確認に使用            |      |
| lint_and_format.sh  | 一通りのLinterとFormatterを実行する |      |
| test.sh             | 一通りのテストを実行する            |      |
| generate_env.sh     | .envの生成                          |      |
| connect_to_awsdb.sh | Bastion経由でAWSのDBにアクセス      |      |
